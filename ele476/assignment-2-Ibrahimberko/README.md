# Assignment-2

In this assignment, you are asked to extend an existing GNU Radio flowgraph. In this repo find `4_1_MakingUpASquareSignal.grc` file. We talked about this GNU Radio example in our class before. It illustrates how to make a square wave using sinusodials. You will extend this flowgraph so that 
* It produces **Triangular** or **Sawtooth** wave in addition to a **square** wave,
* It would **take input** from user to choose the waveform,
* It would produce the waveform by generating fundamental and 4 non-zero **harmonics**.

Name the new flowgraph as `MakingUpAPeriodicSignal.grc`. 

Also, create **a new Python Jupyter Notebook** named `FourierSeriesNotebook.ipynb` that shows your Python code to calculate Fourier Series for periodic waveforms (i.e. Square/Triangular/Sawtooth)

Good luck!
